# CODE_OF_CONDUCT.md
## Code de conduite (version courte, efficace)

Ce dépôt est un laboratoire public : pédagogie, expérimentation, falsifiabilité.
On veut un espace où les gens peuvent proposer, critiquer, tester — sans se faire découper au couteau à beurre.

---

## 1) Comportements attendus
- Être respectueux, même en désaccord.
- Critiquer les idées, pas les personnes.
- Préférer les preuves (tests, logs, métriques) aux affirmations.
- Signaler les erreurs sans humiliation.
- Donner du contexte : “voici ce que j’ai testé, voici ce que j’observe”.

---

## 2) Comportements inacceptables
- Harcèlement, insultes, attaques personnelles.
- Contenu discriminatoire ou menaçant.
- Doxxing (partage d’infos privées).
- Trollage intentionnel, sabotage de discussions.
- Spams répétés ou hors-sujet.

---

## 3) Application
Le mainteneur peut :
- demander une modification de ton,
- fermer une discussion,
- supprimer un contenu,
- bloquer un utilisateur en cas d’abus.

---

## 4) Signalement
Si un problème survient :
- ouvrir une issue “meta” si c’est non sensible,
- sinon contacter le mainteneur en privé.

---

## 5) Philosophie Guasti-friendly
La relecture hostile est bienvenue.
La malveillance, non.
